0. はじめに

　このソフトウェアはDibas32 Ver1.04でNKDibの減色機能を利用するためのプラグイン
です。


1. 著作権／使用条件

　このソフトウェアはフリーウェアであり、使用条件、頒布条件、その他は
「GNU GENERAL PUBLIC LICENSE」に準拠するものとします。
内容については、付属する「COPYING.txt」を参照下さい。


2. 動作環境

　Dibas32 v1.04が動けば大丈夫だと思います。


3. インストール方法

　dbsnkdib.f32を Dibas32 をインストールしたディレクトリにコピーして下さい。


4. アンインストール方法

　Dibas32をインストールしたディレクトリにコピーしたdbsnkdib.f32を削除して下さ
い。


5．使い方

　Dibas32の[減色]メニューに[NkDib's Method (Normal)]と[NkDib's Method (Fine)]
という項目が追加されているはずです。これらを選択することで利用できます。
